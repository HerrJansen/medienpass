# Digitaler Medienpass – Gymnasium Thomaeum

Komplette Lösung mit:
- **Azure Static Web Apps** (Hosting + Auth)
- **Microsoft Entra ID** (Office-365-Schul-Logins)
- **Azure Cosmos DB** (NoSQL)
- **Azure Functions** (Lehrer-API)

## 1. Voraussetzungen
- GitHub Konto
- Azure Subskription
- Microsoft Entra (Schultenant)

## 2. Repository Struktur
Siehe Ordnerbaum in diesem README. Frontend (React + Vite) liegt unter `/frontend`. Funktionen unter `/api`.

## 3. Lokale Entwicklung
```bash
# Frontend
cd frontend
npm install
npm run dev
``
